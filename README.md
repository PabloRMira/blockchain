## How to run it

### In Python

> Prerequisites: conda installed (+ `make` installed for Linux users)

Clone the repository, navigate to the directory with your terminal and use

```
make setup-dev
python main.py
```

### In Typescript

> Prerequisites: npm + node.js installed

Clone the repository, navigate to the directory with your terminal and use

```
npm install
npm build
npm start
```
